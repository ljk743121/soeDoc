Voice of SZSY 点歌系统

深圳实验学校广播站点歌系统

下载源代码快速开始可用网址

如何使用

  • 下载项目源代码，并在项目根目录运行 pnpm install 以安装依赖。

  • 打开快速开始进行配置

  • 在项目根目录下运行 pnpm run dev 以启动开发服务器。

项目功能

该点歌系统整合了用户管理、歌曲管理、歌曲在线搜索及播放、数据批量导出等功能。

歌曲投稿

用户每周刷新可用投稿数

歌曲审核

通过、拒绝和删除歌曲

智能排歌

选定日期区间后根据歌曲数量自动分配

音乐搜索

直接搜索网易云、QQ音乐歌曲进行投稿

在线播放

集成音乐播放器，可在线试听歌曲

音乐源插件扩展

可自行添加其他音乐源

用户管理

支持设置用户可用投稿数、用户封禁、管理权限分配、开放api等

开放时间管理

自定义投稿开放时间段

通知管理

包括查看和发布、删除通知

屏蔽词管理

设置敏感词以屏蔽

自动化

通过开放api获取排歌数据

数据统计

实时更新，可视化展示歌曲数量，歌手和歌曲热度等数据

Markdown支持

编辑通知时可使用Markdown

响应式布局

自动适配桌面端和移动端

深色模式

可以自由切换浅色模式与深色模式

现代化UI

使用TailwindCSS + Shadcn-vue 构建

新功能

用户注册认证

通过学校管理系统获取用户信息

更好的音乐播放器

支持列表循环、随机播放等
